# Руководство пользователя Word Template Generator

Это инструкция для ежедневной работы без технических деталей.

## Что нужно заранее

- Папка `workspace` с:
  - `template.docx`,
  - файлами актов `*.md`,
  - папкой `generated/` (создастся автоматически, если её нет).
- Запущенный генератор через `start.command` (macOS) или `start.ps1` (Windows).

## Основной сценарий (сгенерировать всё)

1. Откройте Word Template Generator.
2. В поле `workspace` укажите путь к папке проекта или нажмите `Выбрать папку...`.
3. Нажмите `Загрузить акты`.
4. Нажмите `Проверить` (рекомендуется).
5. Нажмите `Сгенерировать все`.
6. Готовые DOCX-файлы появятся в папке `generated`.

## Сгенерировать один акт

1. Нажмите `Загрузить акты`.
2. В списке выберите нужный акт.
3. Нажмите `Сгенерировать выбранный`.

## Подсказки по полям и именам файлов

- В полях markdown можно использовать токены `[[...]]`, чтобы подставлять значения других полей.
  - Пример: `описание: "Работы по [[объект]]"`.
- Для дат поддерживается форматирование:
  - `[[дата|dd.MM.yyyy]]` -> `04.03.2026`
  - `[[дата|d MMMMG yyyy г.]]` -> `4 марта 2026 г.`
  - `[[today]]` / `[[сегодня]]` -> сегодняшняя дата в `dd.MM.yyyy`
  - `[[today|d MMMMG yyyy]]` -> сегодняшняя дата в текстовом формате
- Поддерживаемые форматы даты во входных полях:
  - `04.03.2026`
  - `2026-03-04`
  - `04/03/2026`
- `output_name` в акте не обязателен.
  - Если он не задан, имя итогового DOCX будет взято из имени `.md` файла.

## Недавние проекты

- Блок `Недавние проекты` сохраняет последние открытые `workspace`.
- Кнопка `Использовать` подставляет путь и загружает акты.
- Кнопка `Удалить` убирает выбранный путь из истории.
- Кнопка `Очистить все` очищает историю полностью.

История хранится только локально в вашем браузере.

## Где искать ошибки

Внизу интерфейса есть поле `Log`. Если что-то не работает, сначала смотрите туда.

## Частые проблемы

- `Template not found` — в папке нет `template.docx`, или в markdown указан неверный шаблон.
- `No act markdown files found` — в папке нет актов `*.md`.
- `missing values for ...` — в шаблоне Word есть поля, которых нет в markdown.

## Важно

Текущая версия работает в формате **local-first**: генератор должен быть запущен на том же компьютере, где лежит ваш `workspace`.
