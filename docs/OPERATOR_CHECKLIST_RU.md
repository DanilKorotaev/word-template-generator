# Ежедневный чеклист оператора

## Перед началом работы

- [ ] Открыть генератор через `start.command` или `start.ps1`.
- [ ] Убедиться, что выбран правильный `workspace`.
- [ ] Нажать `Загрузить акты`.
- [ ] Нажать `Проверить`.

## Генерация

- [ ] Если нужен весь пакет — нажать `Сгенерировать все`.
- [ ] Если нужен один документ — выбрать акт и нажать `Сгенерировать выбранный`.

## После генерации

- [ ] Проверить, что файлы появились в `generated/`.
- [ ] Открыть 1-2 ключевых DOCX и визуально проверить поля.
- [ ] Закрыть браузер после завершения работы.

## Если возникла ошибка

- [ ] Скопировать текст ошибки из `Log`.
- [ ] Проверить наличие `template.docx` в `workspace`.
- [ ] Проверить корректность YAML front matter в нужном акте.
- [ ] Передать лог и название акта разработчику.
