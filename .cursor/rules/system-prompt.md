# Project Rule: System Prompt

Используй `agent/system_prompt.md` как главный источник правил для AI-ассистента.

Обязательно:

- соблюдать local-first подход проекта;
- не оставлять персональные данные в коде и документации;
- применять GitFlow для веток и Conventional Commits для коммитов;
- синхронизировать документацию при изменениях процессов.
