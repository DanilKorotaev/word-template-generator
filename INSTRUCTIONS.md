# INSTRUCTIONS

Инструкции для Cursor-агента по установке и запуску `word-template-generator` для пользователя без IT-опыта.

## Репозиторий

- Remote: `https://github.com/DanilKorotaev/word-template-generator.git`
- Основная ветка: `main`

## Главный принцип

Пользователь может не иметь установленный Git/Python и не должен вручную работать в терминале.
Cursor выполняет установку, клонирование, обновление и запуск сам.

## Сценарий "Установи проект с нуля"

1. Проверить наличие Git.
2. Если Git отсутствует:
   - Windows: установить `Git for Windows`.
   - macOS: установить через `xcode-select --install` или Homebrew.
3. Клонировать репозиторий в удобную папку.
4. Перейти в папку проекта.
5. Запустить:
   - Windows: `start.bat`
   - macOS: `start.command`
6. Проверить, что Web UI открылся в браузере.

## Сценарий "Обнови и запусти"

1. `git pull origin main`
2. Запуск:
   - Windows: `start.bat`
   - macOS: `start.command`

## Сценарий "Что-то не работает"

1. Попросить пользователя прислать:
   - лог из окна приложения,
   - последние строки терминала.
2. Проверить:
   - есть ли `template.docx` в workspace,
   - корректен ли front matter в `.md`,
   - доступен ли Python 3 (особенно на Windows),
   - не включен ли `python.exe` Store alias.

## Общение с пользователем

- Всегда на русском.
- Простыми короткими шагами.
- Без сложных терминов.
- Если шаг критичен, давать точную кнопку/путь меню.
